# 📺 S905X3 Armbian Home Assistant Core Server (4GB/64GB + External Storage)

Проект превращения ТВ-приставки на чипе **Amlogic S905X3** (4GB RAM / 64GB eMMC) в надежный сервер умного дома. 
Особенность данной сборки — оптимизация для работы с **внешним диском** (USB SSD/HDD), на который выносятся "тяжелые" данные (базы данных, логи, Docker-контейнеры), в то время как система остается на быстрой внутренней eMMC.

## 🛠 Технические характеристики
| Параметр | Значение |
| :--- | :--- |
| **Устройство** | TV Box (S905X3, 4/64GB) |
| **CPU** | Quad-core Cortex-A55 (Amlogic SM1) |
| **RAM** | 4 GB DDR4 |
| **Internal** | 64 GB eMMC (Система) |
| **External** | USB 3.0 SSD/HDD (`/mnt/data`) |
| **OS** | Armbian (Kernel 5.10/5.15 LTS) |
| **ПО** | Home Assistant Core, Docker, Mosquitto |

## 🚀 Преимущества архитектуры
- **Надежность**: Система на eMMC менее подвержена сбоям питания при обновлении ядра.
- **Долговечность**: Логи и БД Home Assistant пишутся на внешний SSD, не изнашивая eMMC.
- **Масштабируемость**: Легкая замена внешнего диска без переустановки ОС.
- **Производительность**: USB 3.0 обеспечивает скорости, достаточные для любых задач HA.

## ⚡ Быстрый старт

### Шаг 1: Подготовка загрузочного носителя
1. Скачайте образ Armbian для S905X3 (рекомендуется версия с ядром **5.10** или **5.15**).
   - Релизы: [ophub/amlogic-s9xxx-armbian](https://github.com/ophub/amlogic-s9xxx-armbian/releases)
   - Рекомендуемый DTB: `meson-sm1-x96-max-plus.dtb` или `meson-sm1-sei610.dtb` (зависит от ревизии платы).
2. Запишите образ на USB-флешку (BalenaEtcher).
3. Вставьте флешку в **синий порт USB 3.0**.
4. Подключите питание, зажав кнопку Reset (спичкой) на 5-10 сек, пока не появится логотип Armbian.

### Шаг 2: Первичная настройка
1. Подключитесь по SSH: `ssh root@192.168.x.x` (пароль обычно `1234`).
