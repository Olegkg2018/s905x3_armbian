# 📺 S905X3 Armbian Home Assistant Core Server

Проект превращения ТВ-приставки на чипе **Amlogic S905X3** (4GB RAM / 32GB eMMC) в энергоэффективный сервер умного дома под управлением **Armbian** и **Home Assistant Core**.

## 🛠 Технические характеристики

| Параметр | Значение |
| :--- | :--- |
| **Устройство** | TV Box (S905X3) |
| **CPU** | Quad-core Cortex-A55 |
| **RAM** | 4 GB DDR4 |
| **Storage** | 32 GB eMMC |
| **OS** | Armbian (Linux) |
| **ПО** | Home Assistant Core (Python venv) |
| **Потребление** | ~2-4 Вт (в простое) |

## 🚀 Преимущества решения
- **Экономичность**: Потребление энергии в разы меньше, чем у мини-ПК или Raspberry Pi с дисками.
- **Тишина**: Полностью пассивное охлаждение (без вентиляторов).
- **Производительность**: 4GB ОЗУ достаточно для комфортной работы HA Core с десятками интеграций.
- **Цена**: Используется доступное б/у оборудование.

## 📂 Структура репозитория
- `README.md` - Документация проекта.
- `scripts/` - Скрипты установки и обновления (планируется).
- `config/` - Примеры конфигурационных файлов Armbian (планируется).

## ⚡ Быстрый старт
1. Скачайте образ Armbian для S905X3 с [официального сайта](https://www.armbian.org/).
2. Запишите образ на SD-карту или eMMC.
3. Настройте сеть и установите зависимости для Home Assistant Core.
4. Следуйте официальной документации HA для установки в виртуальное окружение.
