# 🎙️ Локальный сервер «Алиса» для умной колонки Xiaozhi

В этой папке содержатся файлы конфигурации и плагины для адаптации локального сервера проекта **Xiaozhi AI Voice** под русскоязычного ассистента «Алиса».

## 📂 Структура файлов:
* `agent-base-prompt-ru.txt` — чистый системный промпт для ассистента без китайских иероглифов и китайских символов разметки.
* `data/.config.yaml` — файл переопределения конфигурации (смена голоса на русский женский Svetlana, подключение плагинов, настройка ASR на Groq Whisper и LLM на DeepSeek).
* `plugins_func/functions/` — папка с плагинами:
  * `get_weather_ru.py` — прогноз погоды через бесплатное API Open-Meteo.
  * `get_news_ru.py` — чтение новостей Украины из RSS «Украинской Правды».
  * `web_search.py` — парсинг результатов поиска в Google без API-ключей.
  * `play_music.py` — музыкальный плеер, который ищет песни локально, а при отсутствии — качает аудиодорожку с YouTube через `yt-dlp` и запускает воспроизведение.

## 🚀 Как использовать:
1. Установите оригинальный сервер [xinnan-tech/xiaozhi-esp32-server](https://github.com/xinnan-tech/xiaozhi-esp32-server).
2. Скопируйте файлы из этой папки поверх файлов оригинального сервера:
   * `agent-base-prompt-ru.txt` в корень сервера.
   * `data/.config.yaml` в папку `data/` сервера (создайте ее, если ее нет).
   * Содержимое `plugins_func/functions/` в соответствующую папку плагинов сервера.
3. Пропишите ваши API-ключи Groq (ASR) и DeepSeek (LLM) в файле `data/.config.yaml`.
4. Запустите сервер (через Docker или python) и подключите вашу колонку ESP32-S3 на адрес WebSocket: `ws://[IP_АДРЕС_СЕРВЕРА]:8000/xiaozhi/v1/`.
