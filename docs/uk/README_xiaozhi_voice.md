# 🎙️ Розумна колонка Xiaozhi AI Voice — альтернатива "Аліси" для Home Assistant

Перетворюємо Home Assistant на інтелектуального голосового помічника, який розуміє стан сенсорів вашого будинку та має доступ до інтернету. Повний аналог популярних колонок, але з вашою приватною екосистемою та підтримкою російської та української мов.

## 🚀 Основні можливості

- **Голосовий діалог**: Природне спілкування (STT/TTS) українською та російською мовами.
- **Інтеграція з HA**: Помічник знає все про ваш дім (температуру, стан світла, BMS тощо) і може керувати пристроями.
- **Доступ до знань**: Отримання даних з інтернету (погода, новини, загальні питання).
- **Приватність**: Ви самі керуєте сервісами розпізнавання та синтезу мовлення.

## 🔧 Обладнання

Для проєкту використовується модуль: **ESP32-S3 N16R8 Development Board Xiaozhi AI Voice Dialogue Robot Module**

- **Дисплей**: 0.96 inch OLED
- **Інтерфейс**: Type-C
- **Особливість**: Вбудований мікрофон і аудіовихід.

**Де купити**: [AliExpress](https://www.aliexpress.com/item/1005009398980859.html)

## 🔊 Модернізація (Hardware Mod)

Для отримання якісного звуку рекомендується **вставити пристрій у звичайну комп'ютерну колонку**:

1. Розберіть колонку.
2. Підключіть аудіовихід модуля до підсилювача колонки.
3. Виведіть OLED-екран і мікрофон на корпус.
4. Живлення можна взяти безпосередньо від внутрішнього БЖ колонки через стабілізатор.

Це перетворить дешевий модуль на повноцінну акустичну систему з ШІ-помічником.
