# 📺 S905X3 Armbian Home Assistant Core Сервер (4GB/64GB + Зовнішнє сховище)

Проєкт перетворення TV-приставки на чіпі **Amlogic S905X3** (4GB RAM / 64GB eMMC) на надійний сервер розумного дому.

Особливість цієї збірки — оптимізація для роботи з **зовнішнім диском** (USB SSD/HDD), на який виносяться "важкі" дані (бази даних, логи, Docker-контейнери), тоді як система залишається на швидкій внутрішній eMMC.

---

## 🛠 Технічні характеристики

| Параметр | Значення |
|----------|----------|
| **Пристрій** | TV Box (S905X3, 4/64GB) |
| **CPU** | Quad-core Cortex-A55 (Amlogic SM1) |
| **RAM** | 4 GB DDR4 |
| **Internal** | 64 GB eMMC (Система) |
| **External** | USB 3.0 SSD/HDD (`/mnt/data`) |
| **OS** | Armbian (Kernel 5.10/5.15 LTS) |
| **ПЗ** | Home Assistant Core, Docker, Mosquitto |

---

## 🚀 Переваги архітектури

- ✅ **Надійність**: Система на eMMC менш піддається збоям живлення при оновленні ядра
- ✅ **Довговічність**: Логи та БД Home Assistant пишуться на зовнішній SSD, не зношуючи eMMC
- ✅ **Масштабованість**: Легка заміна зовнішнього диска без перевстановлення ОС
- ✅ **Продуктивність**: USB 3.0 забезпечує швидкості, достатні для будь-яких задач HA

## ⚡ Швидкий старт

### Крок 1: Підготовка завантажувального носія

1. Завантажте образ Armbian для S905X3 (рекомендується версія з ядром 5.10 або 5.15):
   - **Релізи**: [ophub/amlogic-s9xxx-armbian](https://github.com/ophub/amlogic-s9xxx-armbian/releases)
   - **Рекомендований DTB**: `meson-sm1-x96-max-plus.dtb` або `meson-sm1-sei610.dtb`

2. Запишіть образ на USB-флешку (використовуйте BalenaEtcher)

3. Вставте флешку в **синій порт USB 3.0** приставки

4. Підключіть живлення, **затиснувши кнопку Reset** (сірником) на **5-10 секунд**

### Крок 2: Первинне налаштування

1. **Підключіться по SSH**:

```bash
ssh root@192.168.x.x
```

(Пароль зазвичай `1234`)

2. **Оновіть систему**:

```bash
apt update && apt upgrade -y
```

3. **Встановіть Armbian на eMMC**:

```bash
armbian-install
```

Оберіть правильний DTB та дочекайтеся завершення.

4. **Перезавантажте** приставку та вийміть USB-флешку

## 💾 Налаштування зовнішнього диска

Якщо ви плануєте використовувати зовнішній USB SSD/HDD:

1. Підключіть диск до USB 3.0 порту

2. Виконайте скрипт автоматичного налаштування:

```bash
chmod +x prepare_storage.sh
sudo ./prepare_storage.sh
```

Скрипт:
- Відформатує диск в ext4
- Створить точку монтування `/mnt/data`
- Налаштує автомонтування через `/etc/fstab`
