# 🎙️ Умная колонка Xiaozhi AI Voice — альтернатива "Алисе" для Home Assistant

Превращаем Home Assistant в интеллектуального голосового помощника, который понимает состояние сенсоров вашего дома и имеет доступ к интернету. Полный аналог популярных колонок, но с вашей приватной экосистемой и поддержкой русского и украинского языков.

## 🚀 Основные возможности

* **Голосовой диалог**: Естественное общение (STT/TTS) на русском и украинском языках.
* **Интеграция с HA**: Помощник знает всё о вашем доме (температуру, состояние света, BMS и т.д.) и может управлять устройствами.
* **Доступ к знаниям**: Получение данных из интернета (погода, новости, общие вопросы).
* **Приватность**: Вы сами управляете сервисами распознавания и синтеза речи.

## 🛠 Оборудование

Для проекта используется модуль:

## 🔊 Модернизация (Hardware Mod)

Для получения качественного звука рекомендуется **вставить устройство в обычную компьютерную колонку**:
1. Разберите колонку.
2. Подключите аудио-выход модуля к усилителю колонки.
3. Выведите OLED-экран и микрофон на корпус.
4. Питание можно взять напрямую от внутреннего БП колонки через стабилизатор.

Это превратит дешёвый модуль в полноценную акустическую систему с ИИ-помощником.

## 💻 Локальный сервер умной колонки «Алиса» на S905x3 (Armbian)

Если вы хотите развернуть свой собственный приватный сервер на S905x3 (Armbian), который будет полностью заменять оригинальное китайское облако, в репозитории подготовлена сборка в папке [xiaozhi-server/](file:///home/oleg/s905x/s905x3_armbian/xiaozhi-server/).

### 🌟 Основные возможности:
* **Абсолютно без иероглифов**: Исправлена проблема оригинального сервера, из-за которой диктор зачитывал китайские символы. Создан чистый шаблон системного промпта [agent-base-prompt-ru.txt](file:///home/oleg/s905x/s905x3_armbian/xiaozhi-server/agent-base-prompt-ru.txt) и полностью русифицирован плеер.
* **Голосовое перебивание (Barge-in)**: Работает моментально. Как только VAD сервера или кнопка/микрофон клиента фиксируют новую речь пользователя, текущее воспроизведение ответа или музыки останавливается.
* **Новости Украины**: Плагин [get_news_ru.py](file:///home/oleg/s905x/s905x3_armbian/xiaozhi-server/plugins_func/functions/get_news_ru.py) парсит RSS-ленту «Украинской Правды» на русском языке, сообщая последние события.
* **Погода без API-ключей**: Плагин [get_weather_ru.py](file:///home/oleg/s905x/s905x3_armbian/xiaozhi-server/plugins_func/functions/get_weather_ru.py) делает геокодинг любого населенного пункта и запрашивает данные с Open-Meteo.
* **Свободный поиск в Google**: Плагин [web_search.py](file:///home/oleg/s905x/s905x3_armbian/xiaozhi-server/plugins_func/functions/web_search.py) парсит результаты поиска на google.com на русском языке без необходимости в платных API-ключах.
* **Музыкальный онлайн-поиск (YouTube/yt-dlp)**: Плагин [play_music.py](file:///home/oleg/s905x/s905x3_armbian/xiaozhi-server/plugins_func/functions/play_music.py) ищет треки локально в `./music`, а если их нет — в фоновом режиме находит и скачивает аудиодорожку с YouTube по любому произвольному запросу пользователя (например, *«включи спокойную музыку»*, *«включи диско 80-х»*, *«включи песню Алсу Иногда»*), конвертирует её в MP3 и стримит в колонку.

### 📥 Клонирование и быстрый запуск через Docker:

1. **Клонируйте репозиторий на ваш S905x3 с Armbian**:
   ```bash
   git clone https://github.com/Olegkg2018/s905x3_armbian.git
   cd s905x3_armbian/xiaozhi-server
   ```

2. **Запустите Docker контейнер**:
   Сборка поставляется с настроенным `docker-compose.yml`, который автоматически развернет WebSocket-сервер и Веб-панель управления:
   ```bash
   docker-compose up -d
   ```

### 🔌 Сетевые порты и адреса:

* **`http://[IP_АДРЕС_S905X3]:8003/` (Порт 8003 — Панель управления)**
  * Локальный сайт для настройки и мониторинга системы.
  * **Функционал**: отображает текущий статус системы (Ожидание, Запись голоса, Анализ, Говорит), выводит лог чата с бабблами, консоль выполняемых операций и форму изменения настроек в рантайме (выбор голоса EdgeTTS, ввод API-ключей для ИИ, редактирование промпта Алисы, переключение плагинов). Настройки сохраняются в [data/.config.yaml](file:///home/oleg/s905x/s905x3_armbian/xiaozhi-server/data/.config.yaml) и применяются без перезагрузки сервера!
  * Также на порту `8003` работает OTA-сервер для прошивки плат.

* **`ws://[IP_АДРЕС_S905X3]:8000/xiaozhi/v1/` (Порт 8000 — WebSocket-сервер)**
  * Этот адрес необходимо прописать в веб-интерфейсе настроек Wi-Fi/сервера на самой плате ESP32-S3. Порт принимает Opus-аудиопоток с микрофона колонки и стримит синтезированный голос обратно.

### 🗣️ Вейкап-слово (Wake Word) и активация:

1. **На сервере**: В конфигурационном файле [data/.config.yaml](file:///home/oleg/s905x/s905x3_armbian/xiaozhi-server/data/.config.yaml) прописаны русские вейкап-слова: `алиса`, `привет алиса`, `алисочка`. При получении аудиопотока после вейкапа сервер мгновенно переходит в режим анализа.
2. **На клиенте (ESP32-S3)**: Распознавание ключевого слова происходит **локально** на чипе ESP32-S3 силами встроенной библиотеки ESP-SR. 
   * По умолчанию китайская прошивка реагирует на слово «你好小智» (Nihao Xiaozhi) или английское «Hi Xiaozhi». 
   * **Варианты активации**:
     * **Вариант А (Физическая кнопка)**: Нажмите кнопку на плате (или выведите её на корпус колонки), чтобы принудительно разбудить ассистента без произнесения китайских фраз.
     * **Вариант Б (Кастомная прошивка)**: Вы можете пересобрать прошивку ESP-IDF/ESPHome с собственной обученной моделью вейкапа WakeNet под русское произношение слова «Алиса» (подробнее см. в [README_esp32.md](file:///home/oleg/s905x/s905x3_armbian/README_esp32.md) и официальной документации Xiaozhi).
